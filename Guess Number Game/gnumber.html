Guess a Number Game

By: Miguel Tapia

Instructions

Welcome to my Guess a Number Game! To begin, select a minimum and maximum range! Once this range is selected, press the Set Interval Button in order to tell the program to generate a random value within that range.

The range min should appear next to the Min Value, and the range max should appear next to the Max Value. The Random Numbers Selected will be filled after the number is guessed. Provided you continue to play, all randomly generated numbers will continue to appear there! You can try to re-guess these already randomly generated numbers, but I promise, they won't repeat!

Finally, it is time to guess. Select a number within the range. The computer will alert whether the number is too big or too small. Make sure you utilize full value integers (Ex: "5"). If you don't do so, the computer will let you know!

HOPE YOU ENJOY!
